Mehsys Desktop

  • Sistema simples, grátis e open source para agendamento de horários para salão de beleza, barbearia, etc.
  • Banco de dados local, portanto, não precisa de internet para funcionar.
  • Dois níveis de usuário diferentes, 'admin' (acesso completo) e 'user' (acesso bloqueado à partes mais sensíveis do sistema).
  • Geração de relatórios em pdf.

Mehsys Mobile

  • Sistema simples e grátis para agendamento de horários para salão de beleza, barbearia, etc.
  • Banco de dados local, portanto, não precisa de internet para funcionar, mas atenção: (se desistalar o app o banco de dados é excluido junto).
  • Geração de relatórios em pdf, além de ser possível compartilha-los com outros apps.
