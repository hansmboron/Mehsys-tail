Mehsys Desktop

Sistema simples, grátis e open source para agendamento de horários para salão de beleza, barbearia, etc. Banco de dados local, portanto, não precisa de internet para funcionar. Dois níveis de usuário diferentes, 'admin' (acesso completo) e 'user' (acesso bloqueado à partes mais sensíveis do sistema). Geração de relatórios em pdf.
